= Restablecer la contraseña del usuario de la aplicación
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
La contraseña del usuario de la aplicación se utiliza para el registro del proveedor SRA y VASA con vCenter Server.

.Pasos
. Inicie el Administrador de herramientas de ONTAP desde un navegador web: `\https://<ONTAPtoolsIP>:8443/virtualization/ui/`
. Inicie sesión con las credenciales de administrador de las ONTAP tools for VMware vSphere que proporcionó durante la implementación.
. Seleccione *Configuración* en la barra lateral.
. En la pantalla *Credenciales VASA/SRA*, seleccione *Restablecer contraseña*.
. Proporcione una nueva contraseña y confirme la entrada de la nueva contraseña.
. Seleccione *Restablecer* para implementar los cambios.
